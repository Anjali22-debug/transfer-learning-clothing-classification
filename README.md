# Transfer Learning for Clothing Image Classification

This project applies transfer learning using pretrained convolutional neural networks for multi-class clothing image classification.

## Models Compared
- MobileNetV2
- ResNet18

## Dataset
Small clothing image dataset containing 10 clothing categories:
- dress
- hat
- longsleeve
- outwear
- pants
- shirt
- shoes
- shorts
- skirt
- t-shirt

## Project Features
- PyTorch implementation
- Google Colab training
- Transfer learning with pretrained ImageNet weights
- Data augmentation
- Confusion matrices
- Accuracy/loss curves
- Precision, Recall, and F1-score evaluation
- Model comparison visualizations

## Results
ResNet18 achieved the highest classification accuracy, while MobileNetV2 provided a smaller and more lightweight architecture.
